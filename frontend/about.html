What is CampusNav?

CampusNav is an intelligent chatbot that helps users navigate and discover information about Ramdeobaba University. Ask questions naturally — in plain English — and get instant answers about campus locations, facilities, and landmarks.

Unlike a traditional directory, CampusNav understands varied phrasings. Whether you type "where's the library", "what floor is the canteen on", or "tell me about the mess", the system recognizes your intent and returns the relevant information.

How It Works

CampusNav uses a TF-IDF vectorizer to convert user queries into numerical features, which are then classified using a Linear SVM classifier against a structured location database. The model is trained on varied query templates and location aliases to handle natural language flexibility.

The backend is built with FastAPI, serving the ML model over a simple REST API. The frontend is a single-page application with a clean, responsive chat interface.
